四个视频2x2方式排列

一、mp4转nv12
//将mp4转成nv12(yuv420)

ffmpeg -i input.mp4 -an -pix_fmt nv12 -s 1920x1080 output.yuv

//播放nv12

ffplay -video_size 1920x1080 -pixel_format nv12 output.yuv

二、yuv与yuv相互转换
1.nv12转nv21

ffmpeg -pix_fmt nv12 -s 1920x1080 -i nv12.yuv -pix_fmt nv21 nv21.yuv

//播放nv21

ffplay -video_size 1920x1080 -pixel_format nv21 nv21.yuv

2.nv12转YUV420P(I420)

ffmpeg -pix_fmt nv12 -s 1920x1080 -i nv12_1.yuv -pix_fmt yuv420p I420_1.yuv

//播放YUV420P(I420)

ffplay -video_size 1920x1080 -pixel_format yuv420p I420_1.yuv

3.nv12 1920x180转nv12 640x480截取

ffmpeg -pix_fmt nv12 -s 1920x1080 -i nv12_1.yuv -s 640x480 -pix_fmt nv12 640x480.yuv

4.从nv12中提取y、u、v分量

ffmpeg -s 1920x1080 -pix_fmt nv12 -i nv12_1.yuv -filter_complex ‘extractplanes=y+u+v[y][u][v]’ -map ‘[y]’ y.yuv -map ‘[u]’ u.yuv -map ‘[v]’ v.yuv

输出y(1920x1080)、u(960x540)、v(960x540)分量分辨率，用于播放使用：
<1>.Output #0, rawvideo, to ‘y.yuv’:
rawvideo (Y800 / 0x30303859), gray, 1920x1080, q=2-31, 414720 kb/s, 25 fps, 25 tbn, 25 tbc

<2>.Output #1, rawvideo, to ‘u.yuv’:
rawvideo (Y800 / 0x30303859), gray, 960x540, q=2-31, 103680 kb/s, 25 fps, 25 tbn, 25 tbc

<3>.Output #2, rawvideo, to ‘v.yuv’:
rawvideo (Y800 / 0x30303859), gray, 960x540, q=2-31, 103680 kb/s, 25 fps, 25 tbn, 25 tbc

//播放y分量

ffplay -pix_fmt gray -s 1920x1080 y.yuv

//播放u分量

ffplay -pix_fmt gray -s 960x540 u.yuv

//播放v分量

ffplay -pix_fmt gray -s 960x540 v.yuv

5.提取yuv中的y、u、v分量播放

ffplay -s 1280x720 -pix_fmt nv12 -vf extractplanes=’y’ out.yuv

ffplay -s 1280x720 -pix_fmt nv12 -vf extractplanes=’u’ out.yuv

ffplay -s 1280x720 -pix_fmt nv12 -vf extractplanes=’v’ out.yuv

三、从yuv数据中提取N帧视频
<1>.从NV12中提取30帧yuv数据

ffmpeg -s 1920x1080 -pix_fmt nv12 -i nv12.yuv -c:v rawvideo -filter:v select=”gt(n, -1)” -vframes 30 out30.yuv

ffplay -video_size 1920x1080 -pixel_format nv12 out30.yuv

<2>.从NV12中提取1帧yuv数据

ffmpeg -s 1920x1080 -pix_fmt nv12 -i nv12.yuv -c:v rawvideo -filter:v select=”gt(n, -1)” -vframes 1 out_01.yuv

ffplay -video_size 1920x1080 -pixel_format nv12 out_01.yuv

输入为webm文件:
./ffmpeg -c:v libvpx -i ./input.webm -vcodec vp8 -auto-alt-ref 0 -vf scale=”trunc(iw/2/2)*2:trunc(ih/2/2)*2” -acodec copy ./output.webm
如果失败：尝试用vp9解码：
./ffmpeg -c:v libvpx-vp9 -i ./input.webm -vcodec vp8 -auto-alt-ref 0 -vf scale=”trunc(iw/2/2)*2:trunc(ih/2/2)*2” -acodec copy ./output.webm
如果还是失败：
./ffmpeg -i ./input.webm -vcodec vp8 -auto-alt-ref 0 -vf scale=”trunc(iw/2/2)*2:trunc(ih/2/2)*2” -acodec copy ./output.webm

输入为mp4视频：
./ffmpeg -i ./input.mp4 -vcodec libx264 -crf 28 -coder 1 -refs 5 -me_method umh -subq 9 -me_range 32 -trellis 2 -chromaoffset -2 -bf 2 -b_strategy 1 -g 100 -i_qfactor 1.3 -b_qfactor 1.4 -x264opts bframes=5:b-adapt=2:ref=5 -vf scale=”trunc(iw/2/2)*2:trunc(ih/2/2)*2” -acodec copy ./outputmp4

输入为flv文件：
./ffmpeg -i ./xxx.flv -vcodec vp8 -auto-alt-ref 0 -vf scale=”trunc(iw/2/2)*2:trunc(ih/2/2)*2” -acodec copy ./output.webm 转完后再把output.webm手动改成output.flv

1个视频流 + 1个音频流 合成一段视频：
-i F:\ffmpegx64\output.mp4 -i C:\Users\ws\Downloads\沙一汀EL+-+脱离.mp3 -map 0:v:0 -map 1:a:0 -c:v copy -c:a copy 112.mp4

从1个视频中抽取出 视频流 和 音频流：
ffmpeg -i 112.mp4 -c:v copy -an output_video.mp4 -c:a libmp3lame -q:a 4 output_audio.mp3
